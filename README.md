#人脸检测
使用Face++接口实现简单的人脸识别和人脸追踪。

##Face++
Face++是北京旷视科技有限公司旗下的新型视觉服务平台，Face++平台通过提供云端API、离线SDK、以及面向用户的自主研发产品形式，将人脸识别技术广泛应用到互联网及移动应用场景中，人脸识别云计算平台市场前景广阔。

##功能简介
* 人脸识别
* 人脸追踪
